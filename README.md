# Chocolate Sales Analysis

## Overview
An end-to-end analysis of a chocolate sales dataset using three 
industry-standard tools. The goal was to uncover trends in sales 
performance, top products, and regional breakdowns.

## Dataset
- Source: Kaggle
- Contains: Sales transactions including salesperson, country, 
  product, date, amount, and boxes shipped

## Tools Used
| Tool | Purpose |
|------|---------|
| Excel | Data cleaning and pivot table analysis |
| SQL | Querying and aggregating the data |
| Power BI | Interactive dashboard and visualisation |

## Structure
- 📁 [Excel](./Excel) — data cleaning, pivot tables, charts
- 📁 [SQL](./SQL) — queries and results
- 📁 [PowerBI](./PowerBI) — interactive dashboard

## Key Findings
- Identified top performing salespeople by total revenue
- Analysed sales performance across different regions
- Identified best selling products
- Visualised trends using Power BI dashboard
