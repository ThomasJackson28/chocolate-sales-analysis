# Chocolate Sales Analysis

## Overview
An end-to-end analysis of a chocolate sales dataset using three tools — 
Excel, SQL, and Power BI. The goal was to uncover trends in sales performance, 
top products, and regional breakdowns.

## Dataset
- Source: Kaggle
- Contains: Sales transactions including salesperson, country, product, 
  date, amount, and boxes shipped

## Tools Used
| Tool | Purpose |
|------|---------|
| Excel | Data cleaning and pivot table analysis |
| SQL | Querying and aggregating the data |
| Power BI | Visual dashboard and reporting |

## Folder Structure
- 📁 Excel — cleaning, pivot tables, analysis
- 📁 SQL — queries and results
- 📁 PowerBI — dashboard screenshots
